C# Windows Forms POS

This project is a Windows Forms Point of Sale (POS) application built in C# for managing billing, products, and basic sales operations.

📖 About

A simple C# Windows Forms POS system used for handling product sales, billing, and basic inventory management in a desktop application.

🚀 Features
Product management
Sales & billing system
Invoice generation
Basic inventory tracking
Windows Forms UI
Simple CRUD operations
